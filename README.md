# Fesh

Fesh is a shell written in Rust, designed for performance.

## Installation
1. Download the [latest available binary](https://github.com/iamnotmega/fesh/releases).
2. Make the binary executable

### For GUI file managers
Right-click on the executable, select properties and enable the "Executable as a program" setting. Then double-click the binary to run it.

### For terminals
Open your terminal, navigate to the directory containing the binary and then make the binary executable using `chmod`:
```bash
chmod +x fesh
```
Then run the binary like so:
```bash
./fesh
```
